```
flex 3个属性
	flex-grow 定义项目的放大比例 默认是0  不放大
	flex-shrink 定义项目的缩小比例 默认是1 空间不足 缩小
	flex-basis 定义项目所占据的主轴空间,默认是auto 也可以用来设置初始的宽度

flex复合属性 默认取值就是 0 1 auto
	其他取值
		flex: none  ==>  flex: 0 0 auto  不放大不缩小
		flex: auto  ==>  flex: 1 1 auto  放大缩小
		flex: 数字 ==>  flex: 数字 1 0%  放大缩小,数字代表占剩余尺寸份数
		flex: 像素值 ==>  flex: 1 1 像素值  放大缩小,像素值代表子项初始宽度
```

#### 粘性定位

```
概念
	粘性定位，该定位基于用户滚动的位置。
	它的行为就像 position:relative; 而当页面滚动超出目标区域时，它的表现就像 		     	 position:fixed;，它会固定在目标位置。
	
使用
  position: sticky;/* 粘性定位 */
  top: 20px;
  left:0;
```

#### Grid布局

```
网格是一组相交的水平线和垂直线，它定义了网格的列和行。
CSS 提供了一个基于网格的布局系统，带有行和列，可以让我们更轻松地设计网页，而无需使用浮动和定位。

fr 单位
轨道可以使用任何长度单位进行定义。

网格引入了 fr 单位来帮助我们创建灵活的网格轨道。一个 fr 单位代表网格容器中可用空间的一等份。
```
