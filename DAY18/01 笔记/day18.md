# 全局对象GO

```
全局对象(G1oha1 object)
	在浏览器环境中，js引擎会整合所有的script标签中的内容，产生的一个window对象(代表全局环境)
全局变量
    在script标签中声明的变量
全局函数
    在script标签中声明的函数	
关系
    全局变量其实是作为全局对象(Go对象)的一个属性来存在的
    全局函数其实是作为全局对象(Go对象)的一个方法来存在的	

```



# 活动对象AO

```
活动对象(Active object)
    在函数被调用的时候，产生的一个对象，用来保存当前函数内部的执行环境
局部变量
    在函数内部声明的变量
局部函数
    在函数内部声明的函数
关系
    局部变量其实是作为活动对象(AO对象)的一个属性来存在的
    局部函数其实是作为活动对象(AO对象)的一个方法来存在的
```



# 全局预编译

```
概念
	就是在全局环境下(script标签下)，js引擎的预处理方式	
流程
    1,先查找变量的声明，作为Go对象的属性名，值是undefined
    2,在查找函数的声明，作为Go对象的属性名，值是function
    3,从上到下依次执行代码（声明的语句会被略过）
结论
	在全局环境中，如果存在同名的变量和函数，函数的优先级高    
```



# 函数预编译

```
概念
    就是在局部环境下（函数里面），s引擎的预处理方式
流程
    1,在函数被调用的那一刻，就会为当前函数生成一个AO对象
    2,先查找变量和形参的声明，作为AO对象的属性名，值是undefined
    3,使用实参的值，替换形参的值
    4,在查找函数的声明，作为AO对象的属性名，值是function
    5,从上到下依次执行代码（声明的语句会被略过）
结论
	在局部环境中，如果存在同名的变量和函数，函数的优先级高    
```



# 作用域

```
概念
	就是一个区域，用来限定变量在某个范围内起作用(只要是代码，都是存放在一个作用域中)
好处
	可以提高程序的可靠性，减少命名冲突（在ES6语法中，同一个域中不能定义相同的变量，否则报错D
分类
    全局作用域（在全局坏境下）
    	由scrip标签产生的区域，从计算机的角度可以理解为window对象(Go对象)管控的区域
    	全局变量和全局函数都在全局作用域下，在页面关闭的时候销毁
    局部作用域（在函数里面）
    	由函数产生的区域，从计算机的角度可以理解为当前函数对象(Ao对象)管控的区域
		局部变量和局部函数都在全局作用域下，在函数执行完成后销毁
    块级作用域
    	ES6中的语法，存在于if语句，for循环语句中的{}里面，
```



# 作用域链

```
概念
	只要是代码，都是在一个作用域中，写在函数内部就在局部作用域，没有写在函数内部就在全局作用域中
	如果这个函数，内部还有函数，那么在这个作用域中又可以诞生一个新的作用域，就形成了作用域链

作用
	提供了一个查找机制：内部函数可以访问外部函数的变量，使用链式查找来决定哪些数据能被内部函数访问

本质
	本质就是一个数据结构，函数内部可以嵌套函数，每一次嵌套都会形成一个新的作用域，把这些作用域串串起来，就形成了作用域链

```

# 本地存储

```
概念
	Web存储API提供了sessionstorage(会话存储)和local Storage(本地存储)两个存储对象来对网页的数据进行添加、删除、修改、查询操作。

	localStorage用于长久保存整个网站的数据，保存的数据没有过期时间，直到手动去除。
	sessionstorage用于临时保存同一窗口（或标签页）的数据，在关闭窗口或标签页之后将会删除这些数据。

方法
    getItem(keyname)返回指定键的值
    setItem(keyname,value)添加键和值，如果对应的值存在，则更新该键对应的值。
    removeItem(keyname)移除键
    c1ear()清除存储对象中所有的键

特点
    数据存储在用户的浏览器中
    可以方便用户设置或者读取数据，刷新页面不会丢失数据
    容量较大，sessionstorage约5M,localStorage约20M
    只能存字符串，可以将对象JSoN.stringify()转换后存储

sessionstorage(会话存储)
    特点
        1,生命周期为关闭浏览器
        2,在同一个窗口（页面）下数据可以共享
        3,以键值对的形式存储数据  
        
```
