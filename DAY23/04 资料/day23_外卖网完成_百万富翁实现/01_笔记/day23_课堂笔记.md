```
async
	异步的意思,可以将同步函数转成异步函数
await
	等待的意思,等待他后面的promise执行完之后,可以结果属性中的值
好处
	可以简化调用者获取promise执行的结果
	
	async function f(){
		async function fn(){
            return 'aaa'
        }

        const value = await fn()
        console.log(value)
	}
	
	f()
	
事件循环
	概念
		js为解决单线程代码执行的时候不阻塞主进程的一种机制
		
	流程
		1,默认情况下,浏览器会为每一个标签页开启一个新的渲染程序
		2,渲染进程启动之后,会开启一个渲染的主线程,主线程负责执行html,css,js代码
		3,在一开始,渲染的主线程会进入一个无限死循环
		4,每一次循环都会检查消息队列中是否有任务存在,有,就取出第一个任务执行,没有,休眠
		5,其他所有线程都可以随时向消息队列添加任务,新任务会加到消息队列末尾,排队执行
		
	消息队列(常见3种类型)
		1,微队列:用于存放最快执行的任务,优先级最高  Promise.resolve().then()
		2,交互队列:用于存放事件处理的任务,优先级高
		3,延时队列:用于存放计时器时间到了之后的回调函数,优先级一般
```
