#### 浮动

##### 概念

```
概念
	一种布局方式,可以让元素漂浮起来,从而实现块元素的水平布局
作用
	早期用于图文混排,现在主要用于页面布局
语法
	float
		left	向左浮动
		right   向右浮动
		none	默认值,不浮动  在标准流中
```

##### 特点

```
1,浮动元素会脱离标准流(脱标),在标准流中不占位置,相当于从地面飘向了天空
2,浮动元素比标准流级别高,可以覆盖标准流中的元素
3,下一个浮动元素,会在上一个浮动元素的后面进行左右浮动(实现了水平排布)
4,浮动元素会受上面元素边界的影响
5,浮动元素有特殊的显示效果
    行级元素:浮动之后可以设置宽高,相当于变成了块级元素,将来在布局的时候,不需要再加上display:block属性
    块级元素:浮动之后,就不会独占一行,一行可以显示多个,可以实现水平布局
```

##### 清除浮动

```
概念
	因为子元素浮动了,就不能撑开标准流中父元素的高度,就会影响后面元素的布局,需要清除浮动
目的	
	就是为了让父元素有高度,从而不影响后面元素的布局
解决方式一
	直接给父元素设置高度
		特点
			简单粗暴,方便
		弊端
			在某些布局中,父元素的高度可能是由内容决定的,比如新闻列表
解决方式二
	额外标签法
		做法
			1,在父元素内容的最后面添加一个块元素
			2,给这个块元素设置属性 clear:both
		弊端
			会在页面中添加额外的标签,会让页面的结构变的复杂
解决方式三
	单伪元素清除法
		做法
			使用伪元素替代额外标签
		优点
			方便,经常用在项目中,只需要给标签添加clearfix属性就可以清除浮动
解决方式四
	双伪元素清除法
		做法
			添加前后两个伪元素,前面的伪元素用于解决margin塌陷,后面伪元素用于清除浮动
		优点
			比单伪元素新增了解决margin塌陷的问题
```



#### 定位

##### 简介

```
概念
	也是一种布局方式,可以让元素自由摆放在页面中的任意位置(叠加效果)
应用场景
	1,盒子之间的层叠问题,因为定位之后,元素的层级是最高的,可以叠加其他盒子的上面
	2,可以让盒子始终固定在屏幕的某一位置
使用
	1,设置定位的方式:position属性
	2,设置偏移值:水平和垂直方向上就近各取一个
```

##### 相对定位

```
概念
	相对于它之前的位置进行移动(自恋型)
语法
	position:relative
特点
	1,没有脱标,占据空间
    2,参照原点:以前在标准流中的左上角为原点
使用场景
	1,用于小范围的移动
	2,子绝父相(常见布局技巧:可以让子元素在父元素的任意位置上移动)
```

##### 绝对定位

```
概念
	相对于最近有定位的祖先元素进行移动(拼爹型)
语法
	position:absolute
特点
	1,脱标,不占位置
    2,参照的原点:最近有定位的元素
    	父元素没有定位,一层层往上找,如果都没有,就参照body的左上角
使用场景
	子绝父相(常见布局技巧:可以让子元素在父元素的任意位置上移动)
```

##### 子绝父相

```
概念
	一种跟定位相关的布局技巧,子元素是绝对定位,父元素是相对定位
作用
	可以让子元素相当于父元素进行自由移动
好处
	父元素是相对定位的话,就不会脱标,这样对页面的布局影响是最小的
应用场景
	元素水平居中
	元素水平垂直居中
```

##### 固定定位

```
概念
	相对于浏览器进行定位(死心眼定位)
语法
	position:fixed
特点
	1,脱标,不占位置
    2,参照原点:body左上角
应用场景
	让盒子固定在屏幕的某一位置上
```

##### 层级关系

```
不同布局之间的层级关系
	标准流<浮动<定位(定位的优先级是最高的,可以叠加在其他元素之上)
不同定位之间的层级关系
	相对定位,绝对定位和固定定位的优先级是相同的
	写在最下面的元素层级是最高的,会覆盖上面的元素
	可以使用z-index属性更改元素的优先级(值越大优先级越高),这样就不需要修改html的结构
```



#### 其他属性

##### vertical-align

```
概念
	设置一个元素的垂直对齐方式
作用
	可以解决行内元素/行内块元素,垂直对齐问题
取值
	baseline	默认。元素放置在父元素的基线上。
    top	把元素的顶端与行中最高元素的顶端对齐
    middle	把此元素放置在父元素的中部。
    bottom	使元素及其后代元素的底部与整行的底部对齐
```

##### 溢出效果

```
属性名
	overflow
作用
	控制溢出部分内容的显示效果
取值
    visible	默认值。内容不会被修剪，会呈现在元素框之外。
    hidden	内容会被修剪，并且其余内容是不可见的。
    scroll	内容会被修剪，但是浏览器会显示滚动条以便查看其余的内容。
    auto	如果内容被修剪，则浏览器会显示滚动条以便查看其余的内容。
```

##### 隐藏效果

```
使用display:none隐藏:不占位置
使用visibility:hidden隐藏:占据位置
```
