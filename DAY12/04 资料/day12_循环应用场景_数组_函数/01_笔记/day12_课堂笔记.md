#### 循环语句

```
区别
	for循环和while
		每次先判断循环条件,再执行循环体
	do-while循环
		每次先执行循环体再判断条件(不管条件是否成立,都会先执行一次循环体)

应用场景
	循环次数确定,建议使用for循环,因为它的结构比较简单紧凑
	循环次数不确定,一般循环条件是确定的,建议使用while循环
	
continue
	用于跳过循环中的一个迭代。(跳过当前循环,继续下一次循环)
	
break
	用于跳出循环(终止当前循环)
```

#### 数组

```
概念
	存储数据的容器,方便管理数据(对象类型)
作用
	使用单独的变量名来存储一系列的值,并且可以用变量名来访问里面的任意一个值
	数组中的每个元素都有自己的的ID，以便它可以很容易地被访问到(ID一般称之为索引)
	
定义和初始化
	var arr = [] 
	var arr = [1,2,3]
增
	arr[新的索引] = 值
删
	delete arr[要删除的索引]
改
	arr[要修改的索引] = 新值
查
	取值 arr[索引]
	长度 arr.length
遍历
	普通for循环
求和
	遍历并累加
求最值
	遍历并比较
二维数组
	概念
		本质还是一个数组,只不过是数组里面的元素都是一维数组
	取值
		arr[i] 取的是里面的一维数组
		arr[i][j] 取的是里面的一维数组中的元素
	遍历
		双重for循环
```



#### 函数

```
概念
	封装了一段具有特定功能的代码块
作用
	方便用户重复调用
使用
	先定义,然后调用
参数
	就相当于一个媒介,调用者可以通过它将要操作的数据传递到函数内部进行操作,提高了函数的通用性
返回值
	函数可以通过return将操作完的数据返回给调用者,这样调用者拿到数据之后,可以进行其他操作
数组作为函数参数
	形式参数: 任意的变量名 arr
	实际参数:传一个数组名,或者直接将整个数组传进去
```
