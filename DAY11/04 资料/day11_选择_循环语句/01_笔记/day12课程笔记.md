#### 选择语句

```
if语句
	语法
		if(条件表达式){
			语句体
		}
	作用
		主要用于判断一个条件,只有条件成立,才执行对应的代码
		
if-else语句
	语法
		if(条件表达式){
			语句体1
		}else{
			语句体2
		}
	作用
		主要用于判断一个条件,对这个条件成立与否,都会执行相应的代码
		
if-else-if语句
	语法
		if(条件表达式1){
			语句体1
		}else if(条件表达式2){
			语句体2
		}else if(条件表达式3){
			语句体3
		}else if(条件表达式4){
			语句体4
		}
		...
		else{
			语句体n
		}
	作用
		主要用于判断多个条件
		
switch语句
	语法
		switch(表达式或者变量)
        {
            case 值1:
                执行代码块1
                break;
            case 值2:
                执行代码块2
                break;
             ...
            default:
               	前面都不满足的时候执行
               	break
        }
      作用
      	主要用于判断多个条件(基于不同的条件来执行不同的动作)
      	
 区别
 	if和if-else语句 一次只能判断一个条件,但是if-else-if和switch一次可以判断多个条件
 	所有的switch语句都可以使用if-else-if语句进行改写,但是所有的if-else-if语句并不一定能用  switch改写
 	
 应用场景
 	if语句主要用于范围的判断,switch语句主要用于等值判断
 	如果结果只有固定的几个取值,可以考虑使用switch,因为switch层次更加清晰
```



#### 循环语句

```
for循环 
    for(初始化语句;条件判断语句;循环增量语句){
    	循环体体语句
    }

while循环
	基本格式
		while(条件){
			循环体语句
		}
		
	完整格式
		初始化语句
		while(条件判断语句){
			循环体体语句
			循环增量语句
		}

do-while循环
	基本格式
		do{
			循环体语句
		}while(条件)
		
	完整格式
		初始化语句
		do{
			循环体体语句
			循环增量语句
		}while(条件判断语句)
```
