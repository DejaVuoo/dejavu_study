# 流程控制

```
根据代码结构分类
    顺序结构
        从上到下，依次执行
    选择结构
        选择性的执行某些语句
            if语句
            if-else语句
            if-else-if语句
    循环结构
        根据条件重复执行某些代码
			for循环
            while循环
            do-while循环
学习方法
    1,搞清楚每一种结构的语法
    2,搞清楚他们的执行流程
    3,搞清楚他们的应用场景	            
```



# 选择语句

```
if 语句
	语法
		if(条件表达式){
			语句体
		}
	作用
    	主要用于判断一个条件

if-else语句
    语法
    	if(条件表达式){
            语句体1
        }
        else{
            语句体2	
        }
    作用
    	主要用于判断一个条件，对这个条件成立与否. 都会执行相应的代码   	
```
