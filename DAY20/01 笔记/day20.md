# let和const

```
概念
    let用于声明变量，const用于声明常量
    const声明一个只读的常量，一旦声明，常量的值就不能改变
区别
				var				let 			const
作用域			没有块级作用 		有块级作用		  有块级作用
变量提升		  有			 	 没有				没有
能否被修改   	 可以			    可以		不可以（可以修改内容）

应用场景
    建议优先级	 const > let > var
        1,const在编程语言中，一般表示常量，这样阅读代码的人，就不会随意修改，防止误操作
        2,Js引擎对const做了优化，执行效率比较高
    具体使用
		1,声明基本数据类型，如果确定变量后面会被修改，使用let
        	如果确定不修改，使用const,如果不确定修改，可以先用const,后期再改
        2,声明对象类型，比如数组，对象，一般很少修改他们的地址，更多的是修改里面的内容
			所以首选还是const
            
```



# 类和继承

```
概述
    在Es6中，c1ass(类)作为对象的模板被引入，可以通过c1ass关键字定义类。
    class的本质是function。A○
    它可以看作一个语法糖，让对象原型的写法更加清晰、更像面向对象编程的语法。
语法
    class类名{
         constructor(参数1，参数2){
            this.属性名1 = 参数1
            this.属性名2 = 参数2
            }
            方法名（参数列表）{
                方法体
            }
		}
继承
	class A extends B
super关键字
	代表父类，可以调用父类的方法（普通方法和构造函数）

静态成员
    用static修饰的变量，叫做静态属性，只能通过类名调用
    用static修饰的方法，叫做静态方法，只能通过类名调用		
```



# 解构赋值

```
概述
    解构赋值是对赋值运算符的扩展。
    他是一种针对数组或者对象进行模式匹配，然后对其中的变量进行赋值。
    在代码书写上简洁且易读，语义更加清晰明了；也方便了复杂对象中数据字段获取。
语法
	数组模式匹配
		const [变量列表] = 数组
	对象模式匹配
		const {变量列表} = 对象
```

# 数组遍历

```
四种方式
    1,普通for循环：速度最快，可读性差
    2,forEach:速度较快，并且能够控制内容，可以拿到索引和元素来操作
    3,for...of只能获取到元素，获取不了索引
    4,for...in只能获取索引，获取不到元素，可以根据索引获取元素
遍历对象
    for...of
    可以使用解构的方式获取元素里面的属性
    for...in
    可以获取对象的属性名，通过属性名获取属性值
```



# 元素偏移量

```
概念

作用
    1,获取元素距离带有定位父元素的位置（不带单位）
    2,获取元素自身的大小（宽高，不带单位）
```
